# 🌍 [KUSOCIETY](https://kusociety.netlify.app) — Simulador de Civilización

**KUSOCIETY** es un simulador emergente tipo *“God Simulator”* donde una población de humanos evoluciona de forma autónoma dentro de un mundo dinámico generado en una grilla.

El proyecto explora cómo reglas simples pueden generar comportamientos complejos: desde la supervivencia individual hasta la formación de estructuras sociales, roles y asentamientos.

---

## 🧠 Concepto

En KUSOCIETY no controlás directamente a los individuos:  
observás cómo una civilización se desarrolla por sí sola.

Cada humano tiene:
- Necesidades (hambre, energía, salud)
- Comportamientos (explorar, recolectar, socializar)
- Roles emergentes (líder, recolector, explorador, etc.)

El resultado: **complejidad emergente**.

---

## 🚀 Features

- 🧍‍♂️ Simulación de agentes autónomos
- 🌱 Mundo procedural basado en tiles
- 🏗️ Construcción de estructuras (campamentos, etc.)
- ⚔️ Sistema de roles dinámico
- 📊 Estadísticas en tiempo real
- 👁️ Seguimiento de individuos
- ⏱️ Control de velocidad (1x, 5x, 20x, 100x)
- 🧬 Evolución poblacional
